#### Käynnistys:
- Kloonaa repositorio
- Asenna riippuvuudet  
```bash
poetry install
```

- Käynnistä sovellus
```bash
invoke start
```

- Testit

virtuaaliympäristö
```bash
poetry shell
```

- Testien ajo
```bash
invoke test
```
